    # 自定义注解

    注解annotation实事上，Annotation已经被广泛用于各种Java框架，如Spring，Jersey，JUnit (opens new window)，TestNG。注解相当于是一种嵌入在程序中的元数据，可以使用注解解析工具或编译器对其进行解析，也可以指定注解在编译期或运行期有效。这些元数据与程序业务逻辑无关，并且是供指定的工具或框架使用的

    当我们jwt拦截一些不想拦截的接口,可以使用自定义注解

    例如

    定义AuthAccess

    @Target({ElementType.METHOD})
@Retention(RetentionPolicy.RUNTIME)
@Documented
public @interface AuthAccess {
}
