    # 动态菜单和动态路由的实现

    在对菜单路由进行分配时，同样也要对路由进行分配

    实现动态路由

    对后台传过来的数据进行解析导入，例如

    在../router/index.js里对数据解析

    export const setRoutes = () => {
  const storeMenus=localStorage.getItem("menus");
  const homeRoutes={path:"/",component:()=>import("../views/Home.vue"),  redirect:"/login",children:[]}
  //如果有数据
  if(storeMenus)
  {
    const menus=JSON.parse(storeMenus);
    //遍历
    menus.forEach(item=>{
      //如果存在路径
      if(item.path)
      {
        let itemMenus={path:item.path.replace("/",""),name:item.name,component:()=>import("../views/"+item.pagePath+".vue")}
        homeRoutes.children.push(itemMenus);
      }
      //如果有子路由
    else if(item.children.length)
      {
        item.children.forEach(item=>{
          if(item.path)
          {
            let itemMenus={path:item.path.replace("/",""),name:item.name,component:()=>import("../views/"+item.pagePath+".vue")}
            homeRoutes.children.push(itemMenus);
          }
        })
      }
    })
    //加入路由
    router.addRoute(homeRoutes);
  }
}

    实现动态菜单

    同样先获取menus

    menus:JSON.parse(localStorage.getItem("user")).menus,
    1

    使用v-for对路由进行遍历

    <div v-for="item in menus" :key="item.id">
  <div v-if="item.path">
    <el-menu-item :index="item.path">
      <i :class="item.icon"></i>
      <span slot="title">{{item.name}}</span>
    </el-menu-item>
  </div>
  <div v-else>
    <el-submenu :index="item.id">
      <template slot="title">
        <i :class="item.icon"></i>
        <span slot="title">{{item.name}}</span>
      </template>
      <div v-for="subItem in item.children" :key="item.id">
        <el-menu-item :index="subItem.path">
          <i :class="subItem.icon"></i>
          <span slot="title">{{subItem.name}}</span>
        </el-menu-item>
      </div>
    </el-submenu>
  </div>
</div>

    # 遇到的一些bug

    浏览器刷新时，无页面(路由)

    解决

    在router/index.js中调用setRoutes()方法，这样在每次渲染页面时都会重新对路由进行匹配

    用户在第一次启动项目时，直接跳转到404界面

    解决

    在路由导航守卫中写上

     let user=localStorage.getItem("user")
if(!to.matched.length)
{
  let menusRoutes=localStorage.getItem("menus");
  if(menusRoutes)//如果有路由
  {
    next("/404");
  }
  else{//没有则直接跳转到登录界面
    next("/login");
  }
  //其他情况放行即可
  next();
}
