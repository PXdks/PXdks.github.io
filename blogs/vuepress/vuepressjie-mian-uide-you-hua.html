    # 在首页添加一个上下浮动的小月亮

    此时找到.vuepress/theme/components/HomeBlog/index.vue

    首先在 html 模板的

    里添加

    <ModuleTransition delay="0.01">
  <h4 class="scroll-down"></h4>
</ModuleTransition>
    1
    2
    3

    然后在 style 模板中添加样式

    .scroll-down::after {
  //也可以自己设置喜欢的样式
  content: "";
  position: absolute;
  display: block;
  bottom: -5 rem;
  left: calc(50% - 1.5rem);
  width: 2.4rem;
  height: 2.4rem;
  border-radius: 100%;
  border: 0.75rem solid #f8d86a; //月亮黄
  border-left: none;
  border-top: none;
  transform-origin: 50% 50%;
  transform: rotate(45deg);
  -webkit-animation: shaking 3s infinite;
}
//引入animation动画
@keyframes shaking {
  0% {
    transform: translateY(0.7rem);
  }

  50% {
    transform: translateY(-0.9rem);
  }

  100% {
    transform: translateY(0.7rem);
  }
}
